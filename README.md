# Conception et développement d'un progiciel ERP de gestion des achats, des stocks, des flux de transfert et des budgets doté d'un module BI dynamique et d'un assistant conversationnel

Bienvenue dans le dépôt du projet de fin d'études (PFE). 

Ce système ERP (Enterprise Resource Planning) intègre :
- **Gestion des Achats** (Demandes d'achats internes, Bons de commande, Factures, etc.)
- **Gestion des Stocks et Flux de Transfert** (Suivi multi-entrepôts, transferts inter-sites)
- **Gestion des Budgets** (Suivi des enveloppes budgétaires par service)
- **Module BI Dynamique** (Génération de tableaux de bord intelligents en temps réel)
- **Assistant Conversationnel** (Chatbot IA pour faciliter l'interaction avec le système)

## Technologies Utilisées
- **Backend** : Spring Boot 3, Java 21, Spring Security (JWT)
- **Frontend** : React 18, TypeScript, Vite, Tailwind CSS, Recharts
- **Base de données** : PostgreSQL
- **IA & BI** : Intégration Gemini API
- **Déploiement** : Docker & Docker Compose
